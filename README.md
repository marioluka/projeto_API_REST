# Projeto API REST
Este projeto consiste em criar uma API REST para um sistema de uma escola, onde haverá todas as operações CRUD sendo feitas no banco de dados, com sistema de usuários e permissões.
Caso esteja interessado em vê-lo funcionando, faça:
# git pull 
para sua pasta e 
# npm i 
para instalar as dependencias.

Após a preparação, faça:
# npm run dev
para iniciar o app.
